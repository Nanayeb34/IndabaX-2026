Python was first released in 1991 by Guido van Rossum as a successor to the ABC language. Since then, it has grown to become the most popular high‑level programming language, known for its readability and extensive standard library. Key milestones include Python 2.0 (2000) introducing list comprehensions and garbage collection, and Python 3.0 (2008) delivering a major language redesign that dropped several legacy features to improve consistency.
